# 🧮 Math Quiz Game

A simple command-line Math Quiz Game built with Python that tests users on basic arithmetic operations across multiple difficulty levels.

## 📌 Features

- Three difficulty levels
  - **Level 1:** Addition only
  - **Level 2:** Addition and Subtraction
  - **Level 3:** Addition, Subtraction, Multiplication, and Division
- Randomly generated math questions
- Score tracking system
- Input validation using exception handling
- Interactive command-line experience
- Fun way to practice mental math

## 🚀 How It Works

1. Run the program.
2. Choose a difficulty level:
   - `1` → Addition
   - `2` → Addition & Subtraction
   - `3` → Addition, Subtraction, Multiplication & Division
3. Answer the generated math questions.
4. Receive your final score at the end of the game.
